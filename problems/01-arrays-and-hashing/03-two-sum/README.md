# Two Sum

        Given an array of integers nums and an integer target, return the indices of the two numbers that add up to target. Assume exactly one valid pair exists, and you may not reuse the same element.

        LeetCode: https://leetcode.com/problems/two-sum/

        ## Examples
        - Input: `nums = [2,7,11,15], target = 9`; Output: `[0,1]`
- Input: `nums = [3,2,4], target = 6`; Output: `[1,2]`

        ## Constraints
        - 2 <= nums.length <= 100000
- Exactly one valid answer exists.
- Return zero-based indices.
