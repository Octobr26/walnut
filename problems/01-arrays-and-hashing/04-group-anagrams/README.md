# Group Anagrams

        Given a list of strings, group together words that are anagrams of one another. The order of groups and words inside each group does not matter.

        LeetCode: https://leetcode.com/problems/group-anagrams/

        ## Examples
        - Input: `strs = ["eat","tea","tan","ate","nat","bat"]`; Output: `[["bat"],["nat","tan"],["ate","eat","tea"]]`

        ## Constraints
        - Strings may be empty.
- A sorted word or character-count tuple can identify each group.
