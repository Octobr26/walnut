# Encode and Decode Strings

        Design a pair of methods that turns a list of strings into one string and back again, preserving empty strings and delimiter characters.

        LeetCode: https://leetcode.com/problems/encode-and-decode-strings/

        ## Examples
        - Input: `strs = ["neet","co#de",""]`; Output: `["neet","co#de",""]`

        ## Constraints
        - Strings may contain punctuation or digits.
- The encoded format must be unambiguous.
