# Valid Anagram

        Given two strings, return true when they contain the same characters with the same counts, regardless of order.

        LeetCode: https://leetcode.com/problems/valid-anagram/

        ## Examples
        - Input: `s = "anagram", t = "nagaram"`; Output: `true`
- Input: `s = "rat", t = "car"`; Output: `false`

        ## Constraints
        - Inputs are strings.
- A count-based solution should run in linear time.
