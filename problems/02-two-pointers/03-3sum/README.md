# 3Sum

        Return every unique triplet of numbers that sums to zero. Each triplet may be in any order, and the list of triplets may be in any order.

        LeetCode: https://leetcode.com/problems/3sum/

        ## Examples
        - Input: `nums = [-1,0,1,2,-1,-4]`; Output: `[[-1,-1,2],[-1,0,1]]`

        ## Constraints
        - The answer must not contain duplicate triplets.
- Sorting lets the inner search use two pointers.
