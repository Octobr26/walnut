# Two Sum II

        Given a sorted list of integers and a target, return the one-based positions of the two values that add to the target.

        LeetCode: https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/

        ## Examples
        - Input: `numbers = [2,7,11,15], target = 9`; Output: `[1,2]`

        ## Constraints
        - The input is sorted in nondecreasing order.
- Exactly one valid answer exists.
- Return one-based indices.
