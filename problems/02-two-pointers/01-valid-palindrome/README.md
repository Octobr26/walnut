# Valid Palindrome

        Return true when a string reads the same forward and backward after ignoring non-alphanumeric characters and letter case.

        LeetCode: https://leetcode.com/problems/valid-palindrome/

        ## Examples
        - Input: `s = "A man, a plan, a canal: Panama"`; Output: `true`

        ## Constraints
        - Ignore punctuation and spaces.
- Compare letters without case sensitivity.
